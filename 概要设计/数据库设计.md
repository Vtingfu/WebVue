# 数据库详细设计说明书

## 一、用户模块-User

### 1、简介

用于用户注册登录信息。

### 2、表设计

|  属性名   |  类型  |
| :-------: | :----: |
|   Name    | String |
| PassWord  | String |
|  E-mail   | String |
| WeChatNum | String |
| PhoneNum  | String |
|   Right   |  int   |
|   State   |  int   |
| FacePath  | String |

### 3、表描述

**Right：0-管理员；1-普通职员；2-访问者**

**State：0-下线；1-上线**
